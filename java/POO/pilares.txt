classe - planta, modelo, base
objeto - construído, criado, pode ser usado

atributo - tudo que classe/objeto tem de propriedades, caracteristicas
métodos - tudo que classe/objeto tem de funcionalidade, ações, o que ele pode executar sozinho.


caracteristicas----
*objeto:
- indentidade: 2 objetos sempre serão distintos na memória, mesmo que sejam iguais. Distinção feita pela própria existência dos mesmos.
indentificador único(nome da var) para cada objeto.
- Persistência:
tempo de vida, quando será fechado ou deletado.
    -temporário: durante a execução do programa, permanece existente por tempo definido.
    -permanente: armazenado em meio físico(geralmente mem. secundária).


*classe:
Tipo abstrato de dado, um molde: definido, especificado pelo usuário - fábrica de objetos.
=Contem toda a descrição de atributos e métodos


*mensagens - Elemento utilizado para prover a comunicação entre objetos
define:
    - nome do serviço requisitado, o método
    - nome do requisitante, objeto que chamou o método
    - informação do serviço, parâmetros e argumentos


*Atributos :
-em classes
    o atributo é coletivo, comum em todos os objetos criados da mesma classe
-em objetos
    o atributo é individual, cada objeto pode ter um valor diferente para um atributo
    Cada objeto define um valor inicial, que pode coincidir com o valor de outros objetos
    Área de memória(primária/secundária) reservada individualmente para cada objeto

*Métodos: comportamentos. podem manipular dados fora ou dentro do próprio objeto
